# SciVis AI — 科普插圖產生器

> AI-powered science illustration generator for science communication.

🔬 Enter any science topic and instantly generate stunning sci-fi style illustrations.

## Features

- **Instant generation** — Type a topic, get a beautiful science illustration in seconds
- **6 visual styles** — Sci-fi abstract, microscopic, deep space, technical diagram, surreal, nature documentary
- **Flexible aspect ratios** — 1:1, 16:9, 9:16
- **History gallery** — Browse your previously generated illustrations (stored locally)
- **Download & share** — Save images directly or open full resolution

## How It Works

This app calls the [Pollinations.ai](https://pollinations.ai) Image Generation API to create science-themed illustrations from text prompts. All generation happens via the Pollinations API — no backend server needed.

**API used:** `https://image.pollinations.ai/prompt/{prompt}`
